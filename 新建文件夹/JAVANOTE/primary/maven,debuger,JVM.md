
## maven 自动化构建工具

```java
/*
    maven只定义了个生命周期，每个插件可以把自己的工作绑定在上面

    当你执行相应阶段的时候，maven会把前面的阶段都执行一遍（从最开始） 顺便在每个阶段上执行相应的插件目标

    一个阶段上可以绑定若干个目标，执行的顺序取决于你声明的顺序
*/
```
